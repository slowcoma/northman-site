ИНСТРУКЦИЯ ПО ЗАПУСКУ И РЕДАКТИРОВАНИЮ САЙТА

1. ЧТО ВНУТРИ ПАПКИ
- index.html — главная страница сайта
- styles.css — внешний вид сайта
- app.js — логика календаря, модальных окон и формы добавления
- events.js — стартовые мероприятия
- assets/logo.png — место для вашего логотипа
- assets/placeholder-event.png — заглушка фото мероприятия

2. КАК БЫСТРО ЗАПУСТИТЬ САЙТ
Вариант A. Самый простой:
- откройте файл index.html двойным кликом
- сайт откроется в браузере

Вариант B. Через VS Code:
- откройте папку severny_site в VS Code
- установите расширение Live Server (если захотите)
- нажмите правой кнопкой по index.html -> Open with Live Server

Node.js для этой версии сайта не обязателен. Сайт сделан на обычных HTML/CSS/JS.

3. КАК ПОСТАВИТЬ СВОЙ ЛОГОТИП
- подготовьте PNG-файл логотипа
- переименуйте его в logo.png
- замените файл assets/logo.png на свой
- обновите страницу в браузере

4. КАК ИЗМЕНИТЬ МЕРОПРИЯТИЯ БЕЗ КОДА
- откройте сайт
- нажмите на пустую дату -> откроется форма добавления мероприятия
- заполните название, описание, фото, статус, текст кнопки и ссылку
- нажмите Сохранить

ВАЖНО:
Эти изменения сохраняются в вашем браузере через localStorage.
То есть на этом компьютере и в этом браузере они останутся даже после закрытия страницы.
Но если вы перенесете сайт на другой компьютер или браузер, эти изменения туда автоматически не попадут.

5. КАК ИЗМЕНИТЬ СТАРТОВЫЕ МЕРОПРИЯТИЯ НАВСЕГДА
- откройте файл events.js в VS Code
- найдите нужный объект мероприятия
- измените поля:
  date — дата в формате 2026-05-02
  title — название
  description — описание
  image — путь до картинки, например assets/my-event.png
  status — upcoming или past
  buttonLabel — текст кнопки
  buttonUrl — ссылка кнопки

Пример:
{
  date: '2026-05-02',
  title: 'Молодецкая Удаль',
  description: 'Описание мероприятия',
  image: 'assets/placeholder-event.png',
  status: 'upcoming',
  buttonLabel: 'Участвовать',
  buttonUrl: 'https://example.com'
}

6. КАК ДОБАВИТЬ СВОЮ КАРТИНКУ ДЛЯ МЕРОПРИЯТИЯ
- положите картинку в папку assets
- например: assets/event1.png
- затем в поле image укажите:
  assets/event1.png

7. КАК РАБОТАЕТ КНОПКА В КАРТОЧКЕ МЕРОПРИЯТИЯ
- если статус upcoming, обычно пишут: Участвовать
- если статус past, обычно пишут: Отчет о мероприятии
- вы можете задать любой свой текст вручную в поле buttonLabel

8. КАК УДАЛИТЬ МЕРОПРИЯТИЕ
- нажмите на мероприятие
- нажмите Редактировать
- в форме нажмите Удалить

9. ЕСЛИ ХОТИТЕ ВЕРНУТЬСЯ К ИСХОДНЫМ ДАННЫМ
- откройте сайт
- нажмите F12 -> вкладка Console
- вставьте команду:
  localStorage.removeItem('severny-chelovek-events')
- обновите страницу

После этого снова загрузятся стартовые мероприятия из events.js.
